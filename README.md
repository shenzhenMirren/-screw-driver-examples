# 中文说明
本项目是screw-driver的使用示例

本示例的核心内容是 src/test/java 资源文件夹下generate包的CreateClasses.java与CreateStudent.java

通过这两个类创建数据库表与代码,具体可以查看Controller类或测试类中的方法
